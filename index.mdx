---
title: "Logo Soup"
description: "A tiny framework-agnostic library that normalizes and harmonizes logo visuals"
---

<Note>
  **Make logos look good together** — Logo Soup automatically normalizes logo sizes, weights, and visual balance across any framework.
</Note>

## Quick Install

<CodeGroup>
```bash npm
npm install @sanity-labs/logo-soup
```

```bash yarn
yarn add @sanity-labs/logo-soup
```

```bash pnpm
pnpm add @sanity-labs/logo-soup
```

```bash bun
bun add @sanity-labs/logo-soup
```
</CodeGroup>

## Key Features

<CardGroup cols={2}>
  <Card title="Content Detection" icon="magnifying-glass">
    Automatically finds true content boundaries and crops whitespace
  </Card>
  <Card title="Aspect Ratio Normalization" icon="arrows-left-right">
    Balances wide and tall logos using configurable scale factors
  </Card>
  <Card title="Density Compensation" icon="weight-scale">
    Adjusts sizing so bold logos don't overpower light ones
  </Card>
  <Card title="Visual Center Alignment" icon="arrows-up-down">
    Aligns by visual weight, not geometric bounds
  </Card>
  <Card title="Client-Side Processing" icon="laptop">
    No server required — runs entirely in the browser
  </Card>
  <Card title="Framework Agnostic" icon="code">
    Works with React, Vue, Svelte, Solid, Angular, jQuery, and vanilla JS
  </Card>
</CardGroup>
